# Loading Bar and Timer

This is a simple web application that demonstrates a loading bar and a timer. It also includes an image upload and preview feature.

## Features

-   **Loading Bar:** A visual indicator of progress.
-   **Timer:** Measures the time taken for a process to complete.
-   **Image Upload:** Allows users to upload an image.
-   **Image Preview:** Displays a preview of the uploaded image.
-   **Download:** Allows users to download the uploaded image.

## How to Use

1.  Open the `index.html` file in your web browser.
2.  Click on the "Click to upload an image" area to select an image from your computer.
3.  Once an image is selected, a preview will be displayed.
4.  Click the "Free Download" button to start the loading process.
5.  The loading bar will fill up, and the timer will start.
6.  Once the process is complete, the "Download Image" button will appear.
7.  Click the "Download Image" button to download the uploaded image.
